# SQL代码

SmartNoteBook支持运行SQL代码块。

## 创建SQL代码块

创建SQL代码块的两种方法：

* 鼠标移动到代码块的下方，当显示悬浮操作框时，单击`More cell types`，然后选择`SQL`。

![](/assets/sqldm.png)

* 直接单击代码块右上角的 `+` 号或者单元格下方的`Add Code Cell`，然后点击右上角的![](/assets/cvvr.png)，选择`Convert to SQL`。

![](/assets/cvtsqs.png)

## SQL代码的操作

* 选择数据源
* 结果集的名称 `结果将保存为DataFrame`
* 点击执行代码

![](/assets/sqczz.png)
